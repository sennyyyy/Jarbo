# HaGRID gesture priors

`GesturePriors.json` contains normalized numeric hand-landmark features derived from the HaGRIDv2 test annotations. No RGB photographs, personal metadata, image identifiers, or user IDs are included.

Kapitanov, A., Kvanchiani, K., Nagaev, A., Kraynov, R., & Makhliarchuk, A. “HaGRID — HAnd Gesture Recognition Image Dataset.” WACV 2024, pages 4572–4581.
